# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0.0] - 2026-05-25

### Added
- Initial implementation of Deep Q-Learning agent
- 5x5 Grid World environment with Dungeon AI theme
- Manual neural network with forward and backward propagation
- Experience replay buffer
- Epsilon-greedy action selection with decay
- ASCII rendering with fantasy medieval symbols
- Training loop with configurable hyperparameters
- Comprehensive test suite with HSpec
- GitHub Actions CI pipeline
- Complete documentation (README, architecture, DQN theory, neural network explanation)
- GitHub Codespaces compatibility
